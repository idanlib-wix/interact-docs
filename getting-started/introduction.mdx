Interact is a library for creating web animations and effects triggered by user actions such as clicking, hovering, or scrolling. You can use it to build responsive animations that interact with different elements on the page, work across different devices, and conform to accessibility preferences.

<Info>
  Interact comes bundled with `@wix/motion`, its default animation engine.
  However, you can choose to use Interact with other animation engines of your
  choice.
</Info>

## Anatomy of an Interaction

An Interaction defines a cause-and-effect relationship between an action and a visual change on the page. Each Interaction has a [`config` object](ADDLINK) that defines what the interaction does, what triggers it, and which elements it affects.

The Interaction's `config` includes at least the following properties:

- `key`: Unique identifier for the page element the interaction applies to
- `trigger`: User action that initiates the interaction, such as clicking, hovering, or scrolling
- `effects`: Animation or visual change that occurs on the page element when the trigger activates

Learn more about [configuring an interaction](ADDLINK).

## Reusing effects

You can define effects once in the top-level `effects` object, and then reuse them across multiple interactions:

```typescript
import { Interact } from "@wix/interact";

const config = {
  effects: {
    "fade-in": {
      // Define a reusable effect called "fade-in"
      duration: 1000,
      keyframeEffect: {
        name: "fade-in", // Set up its ID
        keyframes: [{ opacity: 0 }, { opacity: 1 }],
      },
    },
  },
  interactions: [
    {
      key: "hero",
      trigger: "viewEnter",
      effects: [{ effectId: "fade-in" }], // Reference the reusable effect by its ID
    },
  ],
};

Interact.create(config);
```

## TypeScript support

Interact provides full TypeScript support. The `InteractConfig` type validates your configuration, including trigger names, effect properties, and more:

<Tabs>
  <Tab title="React">

```typescript
import { Interact, InteractConfig, Interaction } from "@wix/interact/react";

const config: InteractConfig = {
  interactions: [
    {
      key: "hero",
      trigger: "viewEnter",
      effects: [
        {
          duration: 1000,
          keyframeEffect: {
            name: "fade",
            keyframes: [{ opacity: 0 }, { opacity: 1 }],
          },
        },
      ],
    },
  ],
  effects: {},
};

const instance = Interact.create(config);
```
