Interact is a library for creating web animations and effects triggered by user actions on the page. Use it to build responsive animations that work across different elements and adapt to different devices. You can use Interact with vanilla JavaScript, [integrate it with popular frameworks](ADDLINK), or use [pre-made effects](ADDLINK) right out of the box. 

Interact also supports [accessible](ADDLINK) triggers, so your animations work for all users. 

<Info>
  Interact is powered by `@wix/motion`, a core animation engine built on the Web Animations API. Motion comes bundled with Interact, so you don't need to install it separately.
</Info>

## Anatomy of an Interaction

An Interaction defines a cause-and-effect relationship between an action and a visual change on the page. Each Interaction has a [`config` object](ADDLINK) that defines what the Interaction does, what triggers it, and which elements it affects. It has a `key` that selects the element it applies to, a `trigger` that specifies what initiates it, and a list of `effects` that run when the trigger occurs. 

You can also set up [sequences](ADDLINK) for coordinating staggered animations, as well as [conditions](ADDLINK) that determine when Interactions run.

## Reusing effects

To avoid duplication when the same animation applies to different elements or triggers, you can define effects once in the top-level `effects` object, and reference them by their `effectId` across multiple Interactions. 

Learn more about [organizing your configuration](ADDLINK). 

<Tip>
  Instead of writing your own effects using keyframes, you can use [named effects](ADDLINK), which are pre-made animations from the `@wix/motion-presets` package.
</Tip>

## TypeScript support

Interact provides full TypeScript support. Use the `InteractConfig` type to validate your configuration, including trigger names, effect properties, and more. The following example creates a fade-in animation triggered when an element enters the viewport:

<Tabs>
  <Tab title="React">

```tsx
import { useEffect } from "react";
import { Interact, Interaction } from "@wix/interact/react";
import type { InteractConfig } from "@wix/interact/react";

const config: InteractConfig = {
  interactions: [
    {
      key: "hero",
      trigger: "viewEnter",
      effects: [
        {
          duration: 1000,
          keyframeEffect: {
            name: "fade",
            keyframes: [{ opacity: 0 }, { opacity: 1 }],
          },
        },
      ],
    },
  ],
  effects: {},
};

function App() {
  useEffect(() => {
    const instance = Interact.create(config);
    return () => instance.destroy();
  }, []);

  // The <Interaction> component binds the DOM element to the "hero" key.
  return (
    <Interaction tagName="section" interactKey="hero" className="hero">
      <h1>Welcome</h1>
    </Interaction>
  );
}
```
