Quickly set up your first animation with Interact. Here's how to create a hover animation that scales an image:

## Installation

```bash
npm install @wix/interact
```

## Supported frameworks

The `@wix/interact` package provides 3 entry points based on your use case:

<Tabs>
  <Tab title="React">
    When using React:

    ```typescript
    import { Interact, Interaction } from "@wix/interact/react";
    ```

    Learn more about [integrating with React](ADDLINK).

  </Tab>
  <Tab title="Web Components">
    When using Web Components:

    ```typescript
    import { Interact } from "@wix/interact/web";
    ```

  </Tab>
  <Tab title="JavaScript">
    When using vanilla JavaScript:å

    ```typescript
    import { Interact, add, remove } from "@wix/interact";
    ```

  </Tab>
</Tabs>

## Set up an Interaction

<Tabs>
  <Tab title="React">

Use the `Interaction` React component to wrap the element you want to animate.

    <Steps>
      <Step title="Configure the interaction">

    ```tsx
    import React, { useEffect } from "react";
    import { Interact, Interaction, InteractConfig } from "@wix/interact/react";

    const config: InteractConfig = {
      interactions: [
        {
          key: "my-image",
          trigger: "hover",
          effects: [
            {
              keyframeEffect: {
                name: "scale",
                keyframes: [{ scale: 2 }],
              },
              duration: 300,
              easing: "ease-out",
            },
          ],
        },
      ],
      effects: {},
    };
    ```

      </Step>
      <Step title="Create the interaction and render the component">

    ```tsx
    function App() {
      useEffect(() => {
        const instance = Interact.create(config);
        return () => instance.destroy();
      }, []);

      return (
        <Interaction
          tagName="div"
          interactKey="my-image"
          className="image-container"
        >
          <img src="logo.png" alt="Logo" />
        </Interaction>
      );
    }

    export default App;
    ```

    The `useEffect` cleanup method calls `instance.destroy()` to remove the interaction when the component unmounts.

      </Step>
    </Steps>

</Tab>
  <Tab title="Web Components">

Use the `interact-element` custom element to wrap the element you want to animate.

<Steps>
    <Step title="Create the HTML structure">

    Wrap your element in an `<interact-element>` tag and give it a unique `data-interact-key`:

    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Interaction</title>
    </head>
    <body>
        <interact-element
        data-interact-key="my-image">
        <img id="my-image" src="logo.png" alt="Logo" />
        </interact-element>

        <script type="module" src="./main.js"></script>
    </body>
    </html>
    ```
    </Step>

  <Step title="Configure the interaction">

```javascript
// main.js
import { Interact } from "@wix/interact/web";

const config = {
  interactions: [
    {
      key: "my-image",
      trigger: "hover",
      effects: [
        {
          keyframeEffect: {
            name: "scale",
            keyframes: [{ scale: 2 }],
          },
          duration: 300,
          easing: "ease-out",
        },
      ],
    },
  ],
  effects: {},
};

Interact.create(config);
```

  </Step>
  <Step title="Add cleanup">
    Clean up interactions when the page unloads or on SPA navigation:

```javascript
window.addEventListener("beforeunload", () => {
  Interact.destroy();
});
```

  </Step>
</Steps>

  </Tab>
</Tabs>

The image now scales up when you hover over it.
